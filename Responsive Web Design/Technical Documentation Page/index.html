Introduçao

O Flexible Box Module, geralmente chamado de flexbox, foi projetado como um modelo de layout unidimensional, e como um método que poderia oferecer distribuição de espaço entre itens numa interface e capacidades poderosas de alinhamento.

O módulo de layout de caixa flexível facilita o design da estrutura de layout responsivo flexível sem o uso de flutuação ou posicionamento.
